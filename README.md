# projeto-humanas
Este projeto consiste em um site informativo que explica como ingressar nas Forças Armadas brasileiras por meio de concursos públicos. O foco será o concurso da Escola de Formação de Oficiais da Marinha Mercante (EFOMM), apresentando requisitos, etapas do processo seletivo, matérias cobradas e orientações para candidatos interessados.
